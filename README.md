# Herbal Communities

Full-stack Next.js e-commerce and retreat booking site for Herbal Communities.

## Tech Stack

- Next.js 15.5.6
- TypeScript
- Tailwind CSS
- Clerk Auth
- Neon Postgres
- Prisma
- Stripe
- Resend
- Render

## Local Setup

```bash
npm install
npx prisma generate
npx prisma db push
npm run dev
